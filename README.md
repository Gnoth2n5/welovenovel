## Run web on local server
-> run: composer update
-> run: npm update
-> run: php artisan migrate
-> edit .env
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=
    DB_USERNAME=
    DB_PASSWORD=
   
-> run: php artisan serve
-> Tạo Tài khoản
-> Truy cập database đổi role sang admin
-> Login admin để thêm sản phẩm
